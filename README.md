## PHISHER
A GUI app for Facebook phishing :

## Note :
this app is only tested with Kali Linux but can be used as cross platform scince it uses electronjs 


## dependencies :
**Clone repository first :**

**PYTHON 3 :**
pip install flask

**Node.js:**
open terminal
navigate to the cloned directory then
**npm install**
and to start app
**npm start**

## usage:
hit start button until the app throw an forwarded link that can be send out of the network for attack.
once the victim attempt to login the the attempts will be appear in the "Attempts" Field
